A real engine, not a wrapper.

Everything from byte decoding to native code generation lives in Go. Nothing shells out, nothing links C.

Zero cgo

Every layer, from decoder to validator to codegen, is pure Go. No C toolchain, clean cross-compilation, one static binary.

Single-pass runtime

A single-pass x86-64 backend emits native code with microsecond compiles. A typed IR and an optimizing tier are on the roadmap.

Tracked conformance

Every applicable testsuite file runs on its own. The passing frontier is published in SPECTEST.md, not hidden.

Direct memory

mmap-backed linear memory is exposed straight to your code as a Go []byte, with no copies at the host boundary.

Stable host ABI

A stable wrapper ABI is generated for every export, keeping host ↔ wasm calls typed and low-latency across the boundary.

Open source

Apache-2.0 licensed and built in the open. Precompiled .wago blobs give fast reloads through the Go API.

Get started

Up and running in one command

Add wago to your module and embed it, or drive the engine straight from the CLI to run, compile, or validate any .wasm.

  • ✦ Pure Go, installs with go get
  • ✦ Linear memory is just a Go []byte
  • ✦ Precompile to .wago for fast reloads
install 
$ go get github.com/wago-org/wago
cli
$ wago run      fib.wasm 30
$ wago compile  fib.wasm -o fib.wago
$ wago validate fib.wasm
